# pr
- Abusive prayer rejecter
- Defeats communism: Tolstoy says make abusive prayers.
# Build
- For Linux only, tested on version 7.0.0
- For Intel 64 bit only.
- To stop the dots and bangs, edit out the call pdot and call pbang with # on the first column.
- use Make to build the program, ./pr
- needs gcc to link since it calls the C library
- needs as (gas) to assemble it, usually part of the system.
- the pr.service is provided to help set it up as a systemd element.
